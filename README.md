# IR-Blaster-Protocol-V2

Infrared blaster (IR Blaster) Protocol V2.0 aims to create a package that improves drawbacks of previous IR devices. In the need of filtering unwanted traffic signals  and adding metadata like sender and time of signal and creating package-ready for implementation in new manufacturing devices that uses IR sensors, also allowing formation of IoT network.
